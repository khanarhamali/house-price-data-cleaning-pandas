\# House Price Data Cleaning (Pandas)



This project focuses on cleaning and preprocessing a Kaggle house price dataset using Pandas.



\## Key Tasks

\- Handling missing values

\- Cleaning inconsistent categorical data

\- Converting price values like "65 Lac" and "2 Cr" into numeric format

\- Preparing data for analysis or machine learning



\## Project Structure

\- data/ → dataset

\- notebooks/ → Jupyter notebook

\- scripts/ → Python cleaning script
